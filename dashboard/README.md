# SudanPay Dashboard

Admin dashboard for reviewing and approving/rejecting payment transactions.

## Setup

```bash
npm install
cp .env.example .env
npm run dev
```

## Features

- View all pending transactions
- Review payment receipts
- Approve or reject payments
- Real-time status updates
- Transaction history

## Default Credentials

For testing, use the test merchant API key from the backend seed data.
